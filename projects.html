Software Projects

Mapping User Comments back to Newspaper Articles:
We used information retrieval techniques to map readers’ comments on newspaper article back to the article itself in order to automate editors pick and rank the comments based on relevance of the comments. We used a language model to formalize our problem and Apache lucene was used to rank the documents (comments). The code was developed in JAVA and it is available in github.
JSDL++:
We have implemented an upgrade on Job Submission Description Language (JSDL) an extensible XML specification from the Global Grid Forum for the description of simple tasks to non-interactive computer execution systems, called JSDL++ (JSDL version 2.0). JSDL++ can specify multiple job descriptions on one JSDL document which is highly useful in a heterogeneous distributed environment where different end points requires different executables and different set up requirements. In such case, one JSDL document will suffice to make the job executable on all these environments. JSDL++ is already integrated with the production grid middleware Genesis II (developed in-house by UVa Engineers) and used by grid users at University of Virginia, Indiana University and XSEDE.
Tiling Puzzle Solver:
We developed a tiling puzzle solver as a part of the fulfillment of Theory of Computation course requirements. We implemented dancing links to solve arbitrary puzzles using Knuth's algorithm X. You can download the jar from here.
Garments Management Information System:
In this project an automated inventory and production system for a garments factory was implemented. In order to complete the project we followed all the formal steps of software engineering like requirement analysis, requirement modeling, specifying control etc. And the software was implemented in Java Platform with oracle in Back end. Some snapshots of the project are available here.
Implementation of Unix File System & Memory Management System:
Here an efficient and flexible file system for UNIX operating system was implemented. In this project all basic data structures of UNIX file system like inode, super block, boot block etc were used. The system was implemented using UNIX c language. Also a Dynamic Memory Manager was implemented for Unix System. The system was capable of allocating and releasing variable size block and splitting and merging of distributed block dynamically.

Hardware Projects

Energy Efficient Water Heaters:
In the project we designed an efficient tankless water heating system which aims to reduce the energy consumption of a traditional tankless water heater by reducing the stand-by energy and pipe loss of the traditional heaters. This work was later submitted and accepted as a US Patent.
Smart Room:
(http://smartroom.awardspace.com)
In this project a room was interfaced with the computer to control the electrical devices of the room. In the room two modes of control was implemented- autonomous and manual. In autonomous mode the electrical devices are switched on or off based on the number of people on the room and the temperature and the available light. In manual mode the devices can be controlled through voice or blue tooth device or sms. The project was built on .NET(C#) platform. J2ME was used for mobile interfacing and Microsoft speech SDK was used for voice interfacing.
4-bit Microprocessor:
This microprocessor can execute a set of 29 instructions from Intel 80x86 processor family. It had micro programmed control unit with separate data and instruction RAM.
